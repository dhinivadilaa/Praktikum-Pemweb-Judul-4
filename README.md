# Sistem Manajemen Kontak 

Sistem ini adalah aplikasi manajemen kontak dasar yang dibangun menggunakan PHP murni dan Tailwind CSS, serta menggunakan PHP Session sebagai pengganti database untuk menyimpan data kontak dan kredensial login.

##  Tampilan Aplikasi

Berikut adalah tampilan utama dari setiap halaman dalam sistem ini:

---

### 1. Halaman Login

Halaman awal untuk mengakses sistem.

![Tampilan Halaman Login](gambar/Halaman%20Login.png)

---

### 2. Halaman Beranda (Daftar Kontak)

Halaman utama yang menampilkan daftar semua kontak yang tersimpan.

![Tampilan Halaman Beranda (Daftar Kontak)](gambar/Halaman%20Beranda(DaftarKontak).png)

---

### 3. Halaman Tambah Kontak

Formulir untuk memasukkan kontak baru.

![Tampilan Halaman Tambah Kontak](gambar/Halaman%20Tambah%20Kontak.png)

---

### 4. Halaman Edit Kontak

Formulir untuk memperbarui detail kontak yang sudah ada.

![Tampilan Halaman Edit Kontak](gambar/Halaman%20Edit%20Kontak.png)

---

### 5. Halaman Hapus Kontak (Konfirmasi)

Tampilan yang menunjukkan aksi penghapusan kontak (biasanya berupa pop-up konfirmasi atau log).

![Tampilan Halaman Hapus Kontak](gambar/Halaman%20Hapus%20Kontak.png)
